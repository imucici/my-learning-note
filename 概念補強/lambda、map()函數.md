lambda簡介:
-----

* 又被稱為**無名函數**，是種將運算式重複運用的方式，類似函數 (function) ，卻又不像函數需要額外命名函數的識別字 (identifier)。

* lambda `變數名稱`:`運算式`

map()簡介:
-----

map() 會根據提供的函數(function)對指定序列做映射。
第一個参數 function 以参數序列中的每一個元素調用 function 函數，返回包含每次 function 函數返回值的新列表。
